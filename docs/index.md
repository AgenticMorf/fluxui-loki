# Flux UI Loki

FluX UI for Grafana Loki log viewer. Livewire dashboard to query Loki logs from Sail/Docker services.

## Features

- Query Loki logs via Livewire dashboard
- Service dropdown, date/time filters (requires Flux Pro)
- Configurable route, layout, middleware
